# Reflection — AI Prescription Agent

## Overview
The **AI Prescription Agent** project automates reading handwritten medical prescriptions using OCR and AI. It extracts text, understands medicine details, and logs them into Google Sheets while sending updates through Telegram.

---

## Goals
- Automate prescription reading with OCR and NLP.  
- Reduce human errors in interpreting prescriptions.  
- Log and share data efficiently using APIs.

---

## What I Learned
- Using **OCR** to read handwritten text.  
- Connecting **APIs** like Google Sheets and Telegram.  
- Structuring a modular Python project.  
- Handling real-world data accuracy challenges.

---

## Tools Used
Python, Tesseract OCR, OpenAI API, Google Sheets API, Telegram Bot API.

---

## Challenges
- Poor handwriting recognition.  
- Managing API authentication and integration.  
- Handling noisy image backgrounds.

---

## Future Work
- Support for regional languages.  
- Add voice and dashboard features.  
- Improve OCR accuracy with AI models.
